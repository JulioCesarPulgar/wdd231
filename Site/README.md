# Cabimas Chamber — Course Project

This is a small, responsive static website built as a chamber of commerce project for Cabimas, Venezuela. It demonstrates HTML, CSS, and JavaScript focused on accessibility and good UI/UX.

Getting started
- Open `index.html` in your browser for a quick preview.
- For a local static server (recommended), run (Node.js required):

```bash
npx http-server -c-1 .
```

What I added
- `index.html`, `discover.html`, `directory.html`, `join.html` — core pages
- `styles.css` — responsive accessible styles
- `script.js` — nav toggle, directory filters, form validation
- `images/logo.svg`, `images/placeholder.svg` — simple assets
